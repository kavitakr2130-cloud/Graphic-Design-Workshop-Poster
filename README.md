# Thiranex Graphic Design Internship – Layout & Promotional Design

This project focuses on visual communication through the design of a high-impact promotional poster for a Graphic Design Workshop at IIT ISM Dhanbad.

## What's inside
- Marketing layout: A professional A2/Landscape flyer combining text, custom illustrations, and background textures.
- Information hierarchy: Structured to prioritize the event title, followed by workshop topics and registration details.
- Visual depth: Uses subtle geometric background patterns and color overlays to create a modern, high-end feel.
- Technical details: Created using Canva with advanced layering and transparency.
- Key feature: Custom information bar at the bottom for a clear Call-to-Action (CTA).
